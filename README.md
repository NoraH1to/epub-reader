# Epub Reader

based on [`epub.js`](https://github.com/futurepress/epub.js)

## Usage

- first of all

  `$ npm install`

- web site dev

  `$ npm run start`

- desktop site dev

  `$ npm run start:elec`

- web site build

  `$ npm run build`

  output at `dist`  folder

- desktop site build

  `$ npm run build:elec`

  output at `release` folder
